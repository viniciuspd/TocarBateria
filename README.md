# Tocar Bateria

Esse é um projeto simples para praticar Javascript e Typescript.
Nesse projeto foi desenvolvido uma bateria onde você aperta uma tecla do teclado ira fazer um som, com o total de 9 botões, também você pode salvar a melodia que tocou e colocar para tocar novamente.

Tecnologias Utilizadas

- HTML
- CSS
- JavaSript
- TypeScript
